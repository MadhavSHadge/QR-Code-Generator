# QR Code Generator

A simple web-based tool for generating QR Codes from text or URLs. This project allows users to customize the size of the QR Code and download it as a PNG image. It utilizes the [QRCode.js](https://github.com/davidshimjs/qrcodejs) library for QR Code generation.

## Features

- **Input field**: Type any text or paste a URL to generate a QR Code.
- **Size selection**: Choose from various predefined sizes (100x100 to 1000x1000 pixels).
- **Generate and download**: Create and download your QR Code as a PNG image with the click of a button.

### Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/qr-code-generator.git
   ```

2. Navigate to the project directory:
   ```bash
   cd qr-code-generator
   ```

3. Open `index.html` in your web browser.

## How to Use

1. Enter the desired text or URL in the input field.
2. Select the size of the QR Code from the dropdown menu.
3. Click the "Generate" button to create the QR Code.
4. To download the QR Code, click the "Download" button.
